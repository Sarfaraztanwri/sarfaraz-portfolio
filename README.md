# Sarfaraz Tanwri — Portfolio Website

Personal portfolio website of Sarfaraz Tanwri — Graphic Designer & Web Developer based in Karachi, Pakistan.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Add Your Project Images

Place your 6 project images in `public/images/`:
- `brand-identity.png`
- `portfolio-web.png`
- `social-media.png`
- `landing-page.png`
- `print-campaign.png`
- `ui-ux-app.png`

## Deploy to Vercel

1. Push to GitHub
2. Import repo on vercel.com
3. Click Deploy — done!
